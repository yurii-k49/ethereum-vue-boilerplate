## Project setup
```
yarn install
```
Rename .env.example to .env and add your Moralis ServerUrl and ApplicationId
```
VUE_APP_MORALIS_SERVER_URL=YOUR_MORALIS_SERVER_URL
VUE_APP_MORALIS_APP_ID=YOUR_MORALIS_APP_ID
```

### Run Boilerplate
```
yarn serve
```
